# SPDX XML License Parser

Fetches the license using the identifier entered by the user. Prints the license details to the terminal and the licene itself

## Usage

- For Linux/ MacOS:<br>
```./spdx_parser.sh```

- For Win or if the above command doesn't work:<br>
```dart run```
